# DroneHub GCS — ოპერატორის სახელმძღვანელო

DroneHub GCS არის QGroundControl-ის ქართული, DroneHub ბრენდირებული ვერსია. ეს გზამკვლევი ველის ოპერატორებისთვისაა — კავშირი, ფრენა, მისია და პრობლემების მოგვარება.

---

## 1. პირველი გაშვება

1. გახსენით **DroneHubGCS** დესკტოპიდან ან `./tools/run-dhgcs.sh`.
2. პირველ გაშვებაზე macOS შეიძლება მოითხოვოს ქსელის/კამერის ნებართვას — ვიდეოსთვის საჭიროა.
3. ენა ნაგულისხმევად **ქართული**ა; ფონტი: Noto Sans Georgian.

**მნიშვნელოვანი:** თუ აპი დოკში ჩანს, მაგრამ ფანჯარა არ ჩანს — `killall DroneHubGCS` და ხელახლა გაუშვით `./tools/run-dhgcs.sh`.

---

## 2. დრონოსთან დაკავშირება

### რიგი (SITL / სიმულატორი)

1. **ჯერ** გახსენით DroneHub GCS (UDP 14550-ზე უნდა ეძებდეს).
2. **შემდეგ** გაუშვით PX4 SITL ან სიმულატორი.

```bash
./tools/start-sitl-session.sh              # GCS + PX4 (sihsim_quadx)
./tools/start-sitl-session.sh --simulator  # მხოლოდ pymavlink სიმულატორი
```

### რეალური დრონო

- USB / რადიო / Wi‑Fi — აირჩიეთ **პარამეტრები → კავშირები**.
- AutoConnect ჩართულია ხშირ შემთხვევებში; თუ არ უკავშირდება — ხელით დაამატეთ ლინკი.

კავშირის შემდეგ ზედა ზოლში ჩანს: **სისტემის მდგომარეობა**, **რეჟიმი**, **GPS**, **RC**, **ბატარეა**, **ვიდეო**, **ჯანმრთელობა (OK/Warn/Crit)**.

---

## 3. ფრენის ხედი (Fly View)

### ზედა ზოლი

| ელემენტი | რას აკეთებს |
|----------|-------------|
| **DGCS ლოგო** | მარცხენა პანელის გახსნა/დახურვა |
| **სტატუსი** | მზადყოფნა, armed, prearm შეტყობინებები |
| **ინდიკატორები** | რეჟიმი, GPS, RC, ბატარეა, ვიდეო, ჯანმრთელობა |
| **კამერის ხატულა** | მარჯვენა ვიდეო პანელი |

### ქვედა HUD

- **კომპაქტური რეჟიმი (ნაგულისხმევი):** სიმაღლე · სიჩქარე · ბატარეა · სატელიტები + attitude + კომპასი.
- **გაფართოება:** დააჭირეთ ცენტრალურ მეტრიკას — დამატებითი ტელემეტრია (GPS, MGRS, ქარი, მისია…). მდგომარეობა ინახება აპის გადატვირთვის შემდეგ.
- **დრონო არ არის:** ყველა მნიშვნელობა `—` — mock მონაცემები არ ჩანს.

### ფერები (უსაფრთხოება)

- **მწვანე** — ნორმა (ბატარეა >50%, GPS lock, RC კარგი)
- **ნარინჯი** — ყურადღება (ბატარეა 25–50%, სუსტი RC/GPS)
- **წითელი** — კრიტიკული (<25% ბატარეა, კავშირის დაკარგვა, GPS lock არაა)

### მოქმედებები (MAVLink)

- **მოქმედებები** პანელიდან: ფრენა / payload / საშიში კატეგორიები.
- **საშიში ბრძანებები** (parachute, reboot, fence disable, arm/disarm…) — **slide-to-confirm** სავალდებულოა.
- ჯოისტიკზე საშიში მოქმედებები **დაბლოკილია** — გამოიყენეთ Fly View → მოქმედებები.

---

## 4. გეგმის ხედი (Plan View)

- **გეგმიდან გასვლა** — ზედა მარცხენა «გეგმიდან გასვლა».
- ოფლაინ მისია: ნაგულისხმევად **PX4 / MultiRotor**.
- Waypoint-ების დამატება: აირჩიეთ ინსტრუმენტი მარცხენა ზოლში → დააწკაპუნეთ რუკაზე.

---

## 5. ვიდეო

- Toolbar-ში **ვიდეო სტატუსი:** Off / Wait / Live / N/A.
- მარჯვენა პანელი — PiP; სრული ეკრანი — ვიდეო მენეჯერიდან.
- თუ Live არ ჩანს: შეამოწმეთ **პარამეტრები → ვიდეო** და GStreamer build (`./tools/check-video-build.sh`).

## 6. ველის შემოწმების სია (მოკლე)

ფრენამდე:

- [ ] GPS lock + სატელიტები ≥ 6
- [ ] ბატარეა საკმარისი
- [ ] RC ლინკი (თუ გამოიყენება)
- [ ] Prearm / arming checks გავლილი
- [ ] Geofence / RTL განსაზღვრული
- [ ] მისია ატვირთული / შემოწმებული

ფრენის დროს:

- [ ] HUD ცოცხალი ტელემეტრია აჩვენებს
- [ ] საშიში მოქმედებები მხოლოდ დადასტურებით
- [ ] ვიდეო (თუ საჭიროა) Live
